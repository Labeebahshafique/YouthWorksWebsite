# Youth Works Website

A job listing website designed to help students and young individuals find job opportunities in one place.

## Features
- Structured job listings
- Simple and clean user interface
- Responsive design

## Technologies Used
- HTML
- CSS
- SQL 

## Database
This project includes a SQL file (`youth works database.sql`) that contains the structure and sample data for the job listings database.

To use:
1. Import the file into MySQL or any SQL database
2. Run the queries to create tables and insert data
